# FixMatch no CIFAR-10 — Implementação e Avaliação

Este repositório contém a implementação do método FixMatch aplicado ao dataset CIFAR-10 para o assignment da disciplina Deep Learning. O objetivo é explorar técnicas de aprendizado semi-supervisionado, avaliando o desempenho do modelo com diferentes quantidades de dados rotulados.
